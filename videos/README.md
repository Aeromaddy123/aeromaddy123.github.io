# Videos Folder

Place your demo videos here.

## Format requirements:
- Format: MP4 (H.264 codec)
- Resolution: 720p or 1080p
- Length: Under 30 seconds
- File size: Under 10MB each

## Suggested filenames:
- robot-demo.mp4         → Self-Balancing Robot demo
- surveillance-demo.mp4  → AI Surveillance demo

To compress large videos, use HandBrake (free): https://handbrake.fr/
