# Models Folder

Place your .glb files here.

## How to convert SolidWorks → GLB:
1. Open .sldprt in SolidWorks → File → Save As → STL (Fine, Binary)
2. Open STL in Blender (free) → Add material → File → Export → glTF Binary (.glb)

## Naming convention (match projectMedia config in index.html):
- robot.glb        → Self-Balancing Robot
- uav.glb          → Fixed-Wing UAV
- robotic-arm.glb  → 3D-Printed Robotic Arm
- bionic-arm.glb   → Bionic Prosthetic Arm
- quadcopter.glb   → Medical Quadcopter

Keep each file under 5MB for fast loading.
